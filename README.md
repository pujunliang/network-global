## 1. network-global 简介

> 一款基于three.js、glsl等技术实现的开源的可视化网络攻击效果，支持3D、2.5D动态转换，通过shader实现了点特效（扩散、上升等）、飞线、文字动态渲染、投影动态转换效果等。本项目为简单demo版本，支持快速移植到您的项目，可在一定程度上提高您的开发效率，快速实现基于3D canvas的网络态势攻击可视化大屏。
